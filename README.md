# Desafio-Telecom_X2
Desafio Telecom_X2


Segunda Parte del desafio para estudiar Data Science del curso de Alura LATAM.

Se evaluan los datos trabajados en el desafio anterior con la finalidad de aplicar modelos de machine learning para concluir acerca de la cancelación de servicios por parte de clientes de la empresa TelecomX (churn).
